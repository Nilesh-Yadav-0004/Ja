Introduction of JavaScript.

# JavaScript is a high-level, interpreted programming language mainly used to make websites interactive and dynamic.

# It was created by Brendan Eich in 1995 and is one of the core technologies of web development along with HTML and CSS.

IN SIMPLE DEFINITION:- JavaScript is a language that helps you add behavior and logic to web pages.

Example:

* Clicking a button.
* Showing alerts.
* Validating forms.
* Updating content without reloading the page.

# Key Features of JavaScript
🟢 Interpreted Language – Runs directly in the browser
🟢 Dynamic Typing – No need to declare data types
🟢 Event-Based – Responds to user actions
🟢 Lightweight & Fast
🟢 Object-Oriented – Uses objects and prototypes.

#  BASICS OF JavaScript.

1. Variables 

# What is a Variable?

A variable is a container used to store data values in a program.

Example:
        let name = "Nilesh";

 ** Here, name is a variable that stores the value "Nilesh" ** .

 # Types of Variables in JavaScript.

JavaScript provides 3 ways to declare variables:

1️⃣ var (Old method).

Example:- 
        var age = 20;

* Function scoped.
* Can be re-declared and updated.
* Not recommended in modern JavaScript.

2️⃣ let (Modern & Recommended).

Example:- 
        let city = "Mumbai";let city = "Mumbai";

*Block scoped.
*Can be updated.
*Cannot be re-declared in the same scope.

3️⃣ const (Constant values).

Example:- 
        const pi = 3.14;

*Block scoped.
*Cannot be updated or re-declared.
*Must be initialized when declared.

⚖️ Difference Between var, let, const
Feature	var	let	const
Scope	Function	Block	Block
Re-declare	✅ Yes	❌ No	❌ No
Update Value	✅ Yes	✅ Yes	❌ No
Use Today	❌ Avoid	✅ Preferred	✅ Preferred.

🧠 Important Concepts
🔹 Declaration vs Initialization
let x;        // Declaration
x = 10;       // Initialization

# Naming Rules
* Must start with letter, _ or $.
* Cannot use keywords (like let, const).
* Case-sensitive (name ≠ Name).

Example
let name = "Nilesh";
let age = 21;
const country = "India";

console.log(name, age, country);
